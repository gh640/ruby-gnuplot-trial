ruby-gnuplot-trial
==================

Rubyからgnuplotを操作するGnuplot gemを試すためのスクリプト集です。


# 必要なもの
- gnuplot本体
  - http://www.gnuplot.info
- Gnuplot gem
  - https://github.com/rdp/ruby_gnuplot


# 実行環境
- Ruby 2.0.0p247


# 内容
- g01 sin曲線のプロット
- g02 散布図のプロット
- g03 複数の系列の同一グラフへのプロット
- g04 csvからXYデータを読み込んでプロット
- g05 csvからXYデータを読み込んでプロット ついでに近似曲線（1次）を追加
